# Phases 6–10 Planned Features

## Phase 6–7: Daily Plan
- Auto-generate task list based on timetable + free blocks + study goals
- Smart scheduling: prioritize high-credit, neglected courses; fit into free blocks
- Manual task entry; toggle completion with progress tracking
- Generate button on the Plan tab (leftmost in bottom nav)

## Phase 8: Smart Notifications
- Free block alerts (30 min+ gaps)
- Streak-at-risk notifications (8:30 PM if no session today)
- Haven't-studied-yet alerts (8 PM)
- Milestone approaching alerts
- Weekly review prompt (Monday 8 AM)
- Session reminders 10 min before scheduled study tasks
- Toggle settings per notification type

## Phase 9: Smart Insights
- Best study day/hour analysis
- Neglected course warnings
- Study consistency scoring
- Late-night drop detection
- Weekly trend comparison (up/down by %)
- No-data fallback message
- Animated insight cards, grouped by type (warning first)

## Phase 10: Weekly Review
- Monday auto-show (one time per week)
- Accessible from Sessions screen button
- Stats: total hours, best day, current streak, best/neglected course
- Free-text reflection note save
- Optional: past weeks history (back 16 weeks)
